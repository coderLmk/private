    # 为什么要运动？

    如今随着社会压力的增大，人们上了一天班后的疲惫导致大部分人不想占用休息时间来进行运动锻炼，运动量直线下降，大家习惯了久坐，缺乏锻炼，而身体活动量下降，导致身材发胖，各种健康疾病开始出现，人也会加速衰老。
    所以养成运动的好习惯可以强化我们的体能耐力，强化身体机能，肺活量提高了，心脏更加强健了，你的运动能力越来越强，保持旺盛体能状态。运动健身可以促进细胞再生，让皮肤保持紧致状态，抵抗身体的衰老速度，让你颜值保持冻龄状态。

    # 基础代谢是什么？

    基础代谢就是维持我们基本生命的最低热量,简单来说,你今天躺着不动,不思考,不说话,但是身体要维持你活着的状态,例如血液流动、心脏跳动、呼吸等等身体所需要消耗的热量
    基础代谢的消耗占据身体总消耗的60%-70%
